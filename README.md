# Bitasmbl-Developer-Portfolio-Landing-Page-d435a6-b9bbb1a779c7

## Description
A single-page, responsive portfolio landing site for a junior developer featuring hero, about, projects, and contact sections with clean HTML and CSS (optionally Tailwind CSS).

## Tech Stack
- Tailwind CSS

## Requirements
- Create a hero section with developer name, title, and a clear call-to-action button.
- Implement an About Me section with a short paragraph and image placeholder.
- Show three featured projects with title, description, and GitHub link.
- Add a contact form with name, email, and message fields.
- Ensure the page layout is mobile responsive.
- Add simple hover or scroll animations for bonus points.
- Optionally use a CSS framework like Tailwind CSS or Bootstrap.
- Optionally deploy to GitHub Pages or Vercel and include the link in README.

## Implementation Steps
1. Set up `index.html` with semantic sections: hero, about, projects, contact.
2. Include Tailwind CSS in `index.html`.
3. Build the hero with name, title, and call-to-action button.
4. Add About Me section with paragraph and image placeholder.
5. Create three project cards with title, description, and GitHub link.
6. Implement contact form with name, email, and message fields.
7. Apply Tailwind responsive classes for mobile layout.
